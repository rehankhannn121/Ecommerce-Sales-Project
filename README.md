# Ecommerce-Sales-Project
## Project Overview
This project is a Power BI dashboard analyzing e-commerce sales data. 
It provides insights into sales performance, profit trends, and product category analysis.

## Key Features
- Interactive dashboard with filters and slicers
- Profit and sales analysis by category and sub-category
- Trend charts for monthly and yearly sales
- Top-selling products visualization
- User-friendly layout and color theme 

## How to Use
1. Download the `.pbix` file.
2. Open it in Power BI Desktop.
3. Interact with the dashboard using filters and slicers.
